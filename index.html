HTML/CSS: @font-face

Helpful tips to better comprehend the @font-face feature.

What is @font-face?

Definition A special rule that allows font to be used even if the user doesn 't have it installed on their computer.
Make sure to use the font-family and src properties.
src Specifies the path; multiple different paths may be needed (refer to more details in formats below).
format How the font is supplied and this varies by browser (be aware of the license agreements).


@font-face CSS Code Example

@font-face{
font-family:'ChunkFiveRegular'; src:url('fonts/chunkfive.eot'); src:url('fonts/chunkfive.woff'); src:url('fonts/chunkfive.tff'); src:url('fonts/chunkfive.svg');}

Font Formats

Format Description
.eot Stands for embedded open type.
An uncompressed font created by Microsoft.
.woff or .woff2 Stands for web open font format.
Comes in a compressed format and .woff2 is the upgraded version to help increase speed of the web browser.
.tff/otf Stands for TrueType font and OpenType Font.
TFF was created by Apple and has the ability to be resized to any size without losing quality when printed.
OTF was developed by Adobe and Microsoft and are also fully scalable without losing quality.
.svg Stands for Scalable Vector Graphics.
Mainly used to display vector images in XML format.
